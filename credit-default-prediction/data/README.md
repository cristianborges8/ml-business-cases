# Dados

Este diretório deve conter os dados brutos e/ou processados para o projeto de predição de inadimplência.

## Como baixar os dados

1. Defina a fonte de dados (Kaggle, UCI, base interna, etc.).
2. Baixe os arquivos e salve neste diretório.
3. Se houver arquivos grandes, prefira não versionar no Git e documente o caminho no notebook.

## Estrutura sugerida

- `raw/`: dados brutos
- `processed/`: dados tratados
