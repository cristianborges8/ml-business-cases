# Dados

Os dados utilizados neste projeto são públicos e disponibilizados pelo Kaggle.

## Como baixar

1. Acesse: https://www.kaggle.com/c/GiveMeSomeCredit/data
2. Faça login (ou crie uma conta gratuita)
3. Baixe o arquivo `cs-training.csv`
4. Coloque o arquivo nesta pasta (`data/`)

## Arquivos esperados

- `cs-training.csv` — conjunto de treino com ~150.000 registros
- `cs-test.csv` — conjunto de teste (sem label, usado para submissão no Kaggle)

## Atenção

Os arquivos de dados **não estão versionados** neste repositório (estão no `.gitignore`)
para respeitar os termos de uso do Kaggle.
