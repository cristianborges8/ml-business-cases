# Credit Default Prediction

Projeto de machine learning para previsão de inadimplência.

## Estrutura

- `data/`: instruções e arquivos de dados
- `notebooks/`: exploração e modelagem
- `src/`: código-fonte reutilizável
- `outputs/figures/`: gráficos gerados
